# SkillMe
Видеоплатформа для обучения, где весь контент разделен на категории (например: игры, кулинария, музыкальные инструменты и тд). Выбрав определенную категорию, пользователю будут представлены видео по данной тематике, из которых он сможет выбрать понравившееся и обучиться.

A new Flutter project.
